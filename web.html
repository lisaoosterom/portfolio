Eurostar

Microinteracties


Vak

Project WEB

Datum

blok 2 - jaar 2

Categorie

UX / Tech

Tools

Sketch / HTML / CSS / Js

Eurostar wil niet langer bekend staan als een treinen maatschappij, maar als een distribiteur van ervaringen. Voor deze ervaringen zijn de 99 verschillende verhalen in het leven geroepen. Deze verhalen worden op de Homepagina aangeboden aan de gebruiker om voor- na- en tijdens de reis te bekijken.

De opdracht

Aan mij de opdracht om een responsive website te ontwerpen (en coderen) voor de 99 verhalen van Eurostar. Behorende bij deze website ontwerp ik twee micro interacties. Voordat ik deze 2 micro interacties ontwerp doe ik onderzoek naar 2 verschillende (sterke) patterns om mijn eigen micro interactie te ontwerpen.

Concept

Ik heb twee micro interacties voor tijdens het lezen en na het lezen van het verhaal ontworpen. Micro interactie 1: bookmark een verhaal, zodat je hem later kan terug vinden in de verder lezen lijst. Micro interactie 2: een verhaal beoordelen, zodat de andere gebruikers makkelijk leuke verhalen kunnen vinden.

Wat ik heb geleerd van dit project?

In het begin van het proces was het lastig om een sterk ontwerpprobleem voor mezelf op te stellen zodat ik goed pattern onderzoek kon uitvoeren. Nadat dit duidelijk voor mijzelf was ben ik 3 verschillende apps gaan vergelijken met elkaar en een eigen micro interactie gaan ontwerpen. Een belangrijk aspect dat ik bij dit project heb geleerd is het duidelijk documenteren van je onderzoek en resultaten.
